# Course Landing (Next.js + TypeScript + Tailwind)

Scaffolded minimal landing page for a premium online course.

Quick start:

```bash
npm install
npm run dev
```

This project includes:

- Next.js + TypeScript
- Tailwind CSS
- Components: Hero, StatsCard, PricingSplit, Curriculum, ModuleCard

Files of interest:
- pages/index.tsx
- components/Hero.tsx
- components/Curriculum.tsx
- components/PricingSplit.tsx
- styles/globals.css
